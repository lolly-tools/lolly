================================================================================
INTEGRATION POINTS
================================================================================

The font upload feature integrates with:

1. HOST BRIDGE (host.state)
   - Stores font blobs in IndexedDB
   - Key format: font-asset:<assetId>
   - Persistent across sessions

2. FONT REGISTRY (font-registry.ts)
   - Reads fonts from IndexedDB
   - Resolves font families for vector export
   - Handles WOFF2 decompression

3. ENGINE TEXT API (host.text)
   - Uses registry to resolve fonts
   - Passes to HarfBuzz for shaping
   - Produces vector paths

4. UI COMPONENTS
   - FontsManager (Brand Studio)
   - Font pickers in tool inputs
   - Asset catalog
